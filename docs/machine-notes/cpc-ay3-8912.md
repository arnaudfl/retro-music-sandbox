# Amstrad CPC – AY-3-8912 Notes

- CPU: Z80 @ 4 MHz  
- Sound: AY-3-8912  
- 3 channels: A / B / C  
- Waveforms: square + noise  
- Volume: 0–15 (4-bit)  
- No built-in effects

## Guidelines for Strudel

- Max 3 voices simultaneously.
- Use `.s("square")` and noise (`s("white*8")`).
- No modern reverb/chorus.
- Arpeggios can simulate extra voices.
